Feita em perfboard.
Pode ser configurada (jumper) para funcionar com o relógio externo ou interno)

Com a placa com os pinos para baixo

ELI
0== Relogio interno
==0 relógio externo

Os pinos existentes não cobrem o barramento todo. Encaixar com cuidado.
